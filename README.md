# Hackphaistus : 눈 관상을 통한 사용자 능력치 분석 서비스

Hackphaistus🔥는 사용자의 관상을 분석하여, 능력치를 제시해주는 서비스입니다.<br/>
관상이라는 정적인 소재에 사용자의 능력치 분석이라는 컨셉을 추가하여, 트렌디하면서도 색다른 경험을 제공합니다.<br/>
해당 서비스를 통해 나의 사진을 넣고, 능력치를 확인하며 즐거움을 느껴보세요.
